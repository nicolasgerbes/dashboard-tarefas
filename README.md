# Dashboard de Tarefas

Dashboard de tarefas desenvolvido com HTML, CSS e JavaScript, com foco em praticar manipulação do DOM, organização de layout e lógica de movimentação de cards entre colunas.

O projeto simula um quadro de tarefas no estilo Kanban, permitindo visualizar tarefas separadas por etapas de progresso.

## Funcionalidades

- Visualização de tarefas em colunas
- Organização das tarefas por status
- Movimentação dos cards entre colunas
- Interface responsiva e estilizada
- Background personalizado
- Estrutura de arquivos organizada

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
